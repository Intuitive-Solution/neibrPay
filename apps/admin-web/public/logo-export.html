How to Export as PNG:

  1. Right-click on any logo variant above
  2. Select "Inspect Element" or "Inspect"
  3. Find the <svg> element in the inspector
  4. Right-click on the SVG element → Copy → Copy outerHTML
  5. Use an online SVG to PNG converter (like cloudconvert.com)
  6. Or use design tools like Figma, Adobe Illustrator to export

Recommended Sizes:
• Favicon: 32x32px or 64x64px
• Apple Touch Icon: 180x180px
• App Icon: 512x512px
